Começar Gratuitamente Como funciona?
# input.py enviado por você

def processar_vendas(faturamento, ano):
    taxa = 0.15
    return {
        "lucro": faturamento * (1 - taxa),
        "ano": ano
    }
# Instantaneamente online:

$ curl -X POST \
  https://api.script2api.com/run/processar_vendas \
  -H 'Authorization: Bearer seu_token' \
  -d '{"faturamento": 50000, "ano": 2024}'

# Resposta JSON:
{
  "lucro": 42500.0,
  "ano": 2024
}
